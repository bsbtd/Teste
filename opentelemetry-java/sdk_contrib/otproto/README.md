# OpenTelemetry Proto Utils

[![Javadocs][javadoc-image]][javadoc-url]

This module contains code to helps with conversions from OpenTelemetry proto objects to API or SDK
objects (e.g. SpanId, TraceId, TraceConfig etc.).
